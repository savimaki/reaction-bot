ERROR EXPLANATION EXPLANATION
TLD-errorTLD is an abbreviation for Top Level Domain.
This error means you have not specified a TLD.
For example "isitdown example" will return a TLD-error,
because it is missing the ".com, .net etc..." suffix.		Add the missing TLD to your request.
TMR-errorTMR is an abbreviation for Too Many Requests.
This error means you have sent too many requests
within a short period of time.		Wait at least 30 seconds and try again.
HTTP-errorThis error shows up if you added http:// or https://
before your domain. For example "isitdown https://example.com"
is not a valid request, but "isitdown example.com" is.		Remove http:// or https:// prefix from you request.
